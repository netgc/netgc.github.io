集中式日志管理

VMware Aria Operations for Logs

（以前称为 vRealize Log Insight）

通过集中式日志管理、深入了解运维和智能分析功能，大规模管理数据，以便跨私有云、混合云和多云环境进行故障排除和审核。

日志数据中有指导意义的洞察信息

直观了解和智能地查询您的 VMware SDDC、VMware Cloud 与多云环境，以便在检测到问题时提供切实可行的解决方案，并加快根本原因分析。VMware Performance for Log (vRealize Log Insight) 既可以在本地部署环境中使用 (sysin)，也可以作为一项服务跨公有云实现灵活使用和出色的可见性。

  • 快速故障排除和分析

    提供创新型索引编制和基于机器学习的智能分组，从而跨基础架构和应用进行高性能搜索并快速识别异常情况。

  • 采用 DevOps 思维模式

    针对您的应用、基础架构和原生公有云进行运行状况检查 (sysin)，以确保用户的数字化体验。

  • 可扩展的分析

    收集并自动识别机器生成的非结构化日志数据（包括应用日志、网络跟踪、配置文件等）的结构，通过构建高性能索引来执行可扩展的分析。

  • 按照 DevOps 的速度提供日志管理

    了解 DevOps 团队如何克服障碍 (sysin)，提高服务质量和运维效率，并加快根本原因分析。

主要功能特性

  • 通用日志收集和分析

访问 IT 环境中的所有一切（包括操作系统、应用、存储、防火墙、网络设备），以实现企业级可见性。

  • 与 Aria Performance (vRealize Operations) 集成

通过与运维管理功能集成 (sysin)，将运维可见性和主动管理功能延展到基础架构和应用。

  • 内置的 VMware Cloud 知识库

内置对 VMware Cloud 技术的支持，可对整个 IT 环境进行集中分析。

  • 直观的图形用户界面、轻松部署

轻松运行简单的交互式搜索以及深入的分析查询 (sysin)，快速获得洞察信息，从而即时提供价值并提高 IT 效率。

  • 企业级

最近的内部测试发现，Log Insight 在针对 10 亿条日志消息的查询测试中比其他业内领先的解决方案快 3 倍。

  • 灵活使用

选择本地部署许可或 SaaS。独立使用，或作为 Aria Suite (vRealize Suite) 的一部分使用，或通过 Aria Universal (vRealize Cloud Universal) 作为订阅服务使用。

比较各个版本

  • vRealize Log Insight for NSX

    随 NSX 免费提供；仅能用于分析 vSphere 和 NSX 日志数据。

    • vSphere 和 NSX
    • 审核日志 (sysin)

  • vRealize Log Insight

    在本地部署环境中使用，不限制数据。

    • vSphere 和 NSX
    • 其他 VMware 服务
    • 第三方技术

  • vRealize Log Insight Cloud

    以服务形式使用，定价灵活。

    • vSphere 和 NSX
    • 其他 VMware 服务
    • 第三方技术 (sysin)
    • VMware Cloud on AWS、AWS、Microsoft Azure、Google Cloud Platform
    • 日志保留时间 - 30 天
    • 原生日志存档支持

  • vRealize Log Insight Cloud for VMware Cloud on AWS

    随 VMware Cloud on AWS 提供，可以选择升级至完整功能特性。

    • vSphere 和 NSX
    • VMware Cloud on AWS 审核日志内容包
    • VMware Cloud on AWS NSX-T 防火墙日志、VMware Cloud Services Platform (CSP) 审核日志、VMware 本地部署产品日志、非 VMware 日志（公有云、应用、容器）
    • 日志保留时间 - 7 天
    • 1/GB/天
