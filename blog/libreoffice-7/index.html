LibreOffice 7.5 新增功能

The Document Foundation 今天发布了 LibreOffice 7.5 开源、免费和跨平台的办公套件，适用于所有支持的平台，包括 GNU/Linux、macOS 和 Windows，为用户提供新功能和大量改进。

经过近六个月的开发，LibreOffice 7.5 办公套件推出了对深色模式支持的重大改进、比以往更加丰富多彩和充满活力的新应用程序和 MIME 类型图标，以及带有上下文的单一工具栏 UI 的改进版本 - 感知控件，以及支持启动中心按类型过滤文档。

LibreOffice 7.5 通过实施几个新选项继续改进字体功能对话框，在宏编辑器的右下角添加一个新的缩放滑块，在触摸板上实现对旋转和缩放手势的支持 (sysin)，并通过添加几个新选项改进 PDF 导出功能和功能，以及错误修复。

此版本还支持 GTK3 滚动条的 “平滑滚动” 功能、更新的 Sifr 图标集（遵循 GNOME Adwaita 的新圆形样式）、更新的乌克兰语词典和斯洛文尼亚词库、新的波斯语 (fa-IR) 词典以及对宏的支持选择器对话框以记住上次运行的宏。

LibreOffice Write 对书签进行了重大改进，书签现在更加明显，能够将对象标记为装饰性以提高可访问性，新的内容控制类型可提高 PDF 表单的质量，以及工具菜单中新的自动可访问性检查器选项.

此外，Write 现在附带了一个基于 DeepL 翻译 API 的初始机器翻译实现、多个拼写检查改进、一次编辑具有不同设置的多个段落的选项卡的能力，以及几个新的可访问性检查器规则。

LibreOffice Calc 获得了对图表中数据表的支持 (sysin)、通过函数向导中的描述进行搜索的能力、新的 “拼写” 数字格式、不区分大小写的条件格式、改进了使用单个前缀引号 (‘) 输入数字时的行为，以及支持 Kamenický 和 Mazovia 编码。

LibreOffice Impress & Draw 具有一组新的默认表格样式，以及创建和自定义表格样式的能力，支持在导航器中拖放对象，能够在幻灯片中裁剪插入的视频并播放它们，以及支持演示者控制台作为普通窗口运行。

LibreOffice Chart 现在支持数据表，而 LibreOffice Math 现在在侧边栏上提供了元素窗格。

正如预期的那样，LibreOffice 7.5 继续改进与 MS Office 专有文件格式（如 DOCX、PPTX 和 XLSX）的互操作性，试图帮助用户从 MS Office 迁移到 LibreOffice。

“LibreOffice 7.5 针对与 MS Office 共享文档或从 MS Office 迁移的用户提供了大量改进和新功能。这些用户应该定期检查 LibreOffice 的新版本，因为进展如此之快，以至于每个新版本都比前一个版本有显着改进，” — The Document Foundation。

LibreOffice 简介

​

自由免费、全能、跨平台的办公套件

LibreOffice 是一款开放源代码的自由免费全能办公软件，可运行于 Microsoft Windows, GNU/Linux 以及 macOS 等操作系统上。它包含了 Writer, Calc, Impress, Draw, Math 以及 Base 等组件 (sysin)，可分别用于文本文档、电子表格、幻灯片演示文稿、绘图文档、数学公式编辑、数据库管理等工作。

LibreOffice 采用对企业和个人用户均免费的 MPL 2.0 授权协议。您可以自由分发该软件，无需支付授权费用（但您仍然可以付费获得经认证的专业支持）。它的源代码完全公开，任何人都可以参与软件的开发和维护。

支持各种文档格式

LibreOffice 支持各种文档格式。除了它原生支持的开放文档格式 (OpenDocument Format, ODF) 外，它还支持许多的非开放格式，比如微软的 Microsoft® Word, Excel, PowerPoint 以及 Publisher 的格式等。

可扩展性

LibreOffice 支持扩展功能。您可以使用 LibreOffice Basic, Python, C/C++, Java 等多个编程语言为 LibreOffice 开发扩展程序，以丰富其功能。LibreOffice 的扩展软件库中已经有多个扩展可用 (sysin)。

另外，它还支持模板功能，您可以使用 LibreOffice 模板库中已有的模板，或者创建自己的模板。

基于强大的基因

LibreOffice 的前身是 OpenOffice.org, 而 OpenOffice.org 最早是基于 StarOffice。它经过了许多年的开发和积累，并已被全球范围内的众多用户使用。今天，LibreOffice 是目前最活跃的 OpenOffice.org 代码库后继者，每 6 个月发布一个主要的新版本。
