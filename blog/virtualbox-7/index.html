VirtualBox 简介

VirtualBox 是一款功能强大的 x86 和 AMD64/Intel64 虚拟化产品，适用于企业和家庭使用。VirtualBox 不仅是面向企业客户的功能极其丰富的高性能产品，它还是唯一一款根据 GNU 通用公共许可证 (GPL) 第 3 版条款作为开源软件免费提供的专业解决方案 (sysin)。请参阅 “关于 VirtualBox” 中的介绍。

目前，VirtualBox 可在 Windows、Linux、Macintosh 和 Solaris 主机上运行 (sysin)，并支持大量 客户操作系统，包括但不限于 Windows（NT 4.0、2000、XP、Server 2003、Vista、Windows 7、Windows 8、Windows 10)、DOS/Windows 3.x、Linux（2.4、2.6、3.x 和 4.x）、Solaris 和 OpenSolaris、OS/2 和 OpenBSD。

VirtualBox 正在积极开发并频繁发布，并且具有不断增长的功能列表、支持的客户操作系统和运行它的平台。VirtualBox 是由一家专门的公司支持的社区杰作：鼓励每个人做出贡献，而 Oracle 确保产品始终符合专业质量标准。

新增功能

VirtualBox 7.1.0（2024 年 9 月 9 日发布）

这是一个重大更新。添加了以下新功能：

  • GUI：现代化的外观和感觉，提供基本和经验丰富的用户级别之间的选择，并具有减少或完整的 UI 功能
  • OCI 集成：性能仪表板现在显示云虚拟机的资源使用情况
  • OCI 集成：克隆计算实例
  • OCI 集成：重置计算实例
  • Oracle VirtualBox Extension Pack：更新到 PUEL 许可证，删除评估选项。要评估使用扩展包，请从 Oracle Software Delivery Cloud 下载它
  • VRDE：如果用户未使用自定义证书设置 TLS，请使用自签名证书启用它 (sysin)，包括在旧证书过期之前颁发新证书
  • NAT：支持 IPv6 的新引擎
  • macOS/Arm 主机：适用于 Linux 和 BSD VM 的 Arm 虚拟化
  • Linux 主机和客户机 (sysin)：添加了对剪贴板共享的 Wayland 支持（错误 ＃20808 ）
  • 共享剪贴板：添加了对在 Linux 和 Windows 主机 / 来宾上传输文件的初始支持。这需要安装 7.1 Guest Additions。有关已知限制，请参阅用户指南
  • 屏幕录制：编码管道的性能得到显着改进，从而减少录制时的 CPU 使用率。这需要在 Windows 客户机上安装 7.1 Guest Additions

此外，还修复和 / 或添加了以下项目：

  • VMM：修复了使用 Hyper-V 时 CPUID 指令的 EFLAGS.TF 处理（错误 ＃17961 ）
  • virtio-net：FreeBSD 12.3 和 pfSense 2.6.0 的后续修复（bug #21201 ）
  • EFI：能够将新的 Microsoft DB/KEX 证书添加到新创建的 VM
  • Oracle 扩展包：修复了完整 VM 加密的加密支持模块的发布
  • GUI：改进了整体可访问性
  • GUI：更新所使用的 Qt 版本
  • 无人值守安装：添加了对基于 subiquity-/cloud-init 的安装程序的支持 (sysin)；这使得能够在无人值守模式下安装更现代的基于 Linux 的来宾操作系统
  • 无人值守安装：添加了对为用户和管理员 / 根帐户指定单独密码的支持
  • VBoxManage：添加了显示和（重新）附加到虚拟机的（视频）记录进度的功能
  • VBoxManage：添加了通过新的 “mount” 访客控制子命令查询访客安装点的功能。至少需要 7.1 来宾添加
  • Python：我们针对 Python 2.x 的 API 绑定现已标记为已弃用，并将在未来版本中删除。请升级您的代码以使用 Python 3
  • VBoxShell：许多更小和更大的错误修复 + Python 3.x 兼容性
