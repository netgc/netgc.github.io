AlmaLinux，一个开源和永远免费的企业 Linux 发行版和社区。

由社区管理和驱动，专注于长期稳定性并提供与 RHEL 二进制兼容的强大生产级平台。

注：既 Redhat “闭源事件” 之后，AlmaLinux OS 基金会董事会决定放弃与 RHEL 1:1 的目标。AlmaLinux 操作系统的目标是与 RHEL 二进制兼容。

由成熟的 CloudLinux OS 背后的团队创建。

为什么是 “Alma”？ 社区是 Linux 的灵魂——为了向社区致敬，我们决定将我们的新发行版称为 Alma：拉丁语中的灵魂一词。

AlmaLinux OS 8（基于 AlmaLinux 8）现已可用。AlmaLinux OS 的第一个稳定版本于 2021 年 3 月 30 日推出。

关于 AlmaLinux 操作系统

AlmaLinux OS 是一个开源的、社区驱动的 Linux 操作系统，它填补了 CentOS Linux 稳定版本停产留下的空白。AlmaLinux OS 是 RHEL® 的 1:1 二进制兼容分支。由社区引导和建设。

作为一个独立的、完全免费的操作系统，AlmaLinux OS 享有 CloudLinux Inc 每年 100 万美元的赞助和其他赞助商的支持 (sysin)。正在进行的开发工作由社区成员管理。

AlmaLinux OS 基金会是一个 501©(6) 非营利组织，旨在为 AlmaLinux OS 社区带来利益。

特性

稳定、生产就绪、基于 AlmaLinux

AlmaLinux OS 是一个企业级服务器操作系统和一个稳定的 Linux 发行版，定期发布，支持窗口很长。您可以依靠 AlmaLinux OS 来运行任何关键工作负载。

  • CentOS 替代品和精确的 AlmaLinux 克隆

  • 生产就绪且稳定，匹配 AlmaLinux 功能 1:1

  • 与 AlmaLinux 同步发布的稳定版本和更新的可预测性

从 CentOS 轻松迁移

切换发行版通常既昂贵又耗时，但从 CentOS 切换到 AlmaLinux 操作系统时并非如此。切换很容易，几乎是瞬间的。

  • 从 CentOS 轻松过渡 (sysin)，只需最少的投资且无需更改软件

  • 从 CentOS 过渡到 AlmaLinux 操作系统的简单迁移脚本

  • 零迁移停机时间。即时迁移甚至大型服务器群

完全支持，永远免费

AlmaLinux OS 基金会成员、赞助商和合作伙伴通过投资和长期支持承诺支持 AlmaLinux OS，以确保分发不受限制、费用和收费。

  • 永远免费和开源：没有许可证，没有使用限制。可自由重新分发

  • 依靠社区拥有和管理的项目

  • 8.x 发布到 2029 年的支持承诺

关键问题的解答

AlmaLinux 操作系统适用于谁？

目前依赖 CentOS 稳定版本来实现计算目标的任何人 (sysin)。需要类似 AlmaLinux 的企业级发行版但不想或无法支付 AlmaLinux 许可证费用的个人和组织。

为什么我们称它为 AlmaLinux OS？

Alma 在西班牙语和其他拉丁语中的意思是灵魂。回顾过去，很容易看出 Linux 如此流行和普及，这要归功于充满热情、多元化的开发人员社区的努力。这个社区是 Linux 的灵魂，每个依赖 Linux 发行版的人都感谢 Linux 社区的努力。这就是我们将新发行版称为 AlmaLinux OS 的原因。

是什么激励了我们？

我们已经这样做了 - 10 年。我们拥有员工、经验、能力和资源。

如何保护社区免受未来课程变化的影响？

该项目由非营利组织拥有和控制，社区参与每一步，包括管理委员会。由于一切都是开放的，社区可以随时决定对项目进行任何更改。

为什么我应该依赖一个由 CloudLinux, Inc. 成员组成的团队来替代 CentOS？

你不是。该项目背后有一群人，而 CloudLinux 只是其中之一。CloudLinux 的核心产品 CloudLinux OS 是一个 AlmaLinux 分支，已经存在了十多年。包括 Dell、Liquid Web 和 1&1 在内的 4,000 多家公司在超过 200,000 个产品安装中依赖 CloudLinux 操作系统。CloudLinux 在创建和维护 AlmaLinux 分支方面拥有丰富的经验 (sysin)，从 AlmaLinux 第 5 版开始一直到第 8 版。AlmaLinux OS 是我们将我们在 AlmaLinux 方面的专业知识转化为服务于更广泛社区的 Linux 发行版的机会。此外，我们从 AlmaLinux OS 一开始就包括了 Linux 社区。展望未来，社区成员将成为 AlmaLinux OS 项目的管理委员会成员，并参与关键决策。最后，AlmaLinux OS 将永远是免费和开源的。社区可以随时选择并继续开发 AlmaLinux OS。

我们计划何时发布 AlmaLinux 操作系统？

在交付 AlmaLinux 操作系统时，我们实际上是在继续一个已经存在的过程，因为我们在分叉 AlmaLinux 方面的经验。我们有信心快速交付 AlmaLinux 操作系统，并计划在 2021 年第一季度交付。

我想从 CentOS 转换为 AlmaLinux 操作系统。我该怎么做？

切换 Linux 发行版可能会令人头疼，但从 CentOS 切换到 AlmaLinux 操作系统时并非如此。就像 CentOS 一样，AlmaLinux OS 是 AlmaLinux 的 1:1 二进制兼容分支，因此切换很简单。一旦 Beta 发布，我们将提供交换存储库和密钥的说明。

我真的可以用一个命令将整个服务器从 CentOS 移植到 AlmaLinux OS 吗？

如前所述，AlmaLinux OS 将是 AlmaLinux 的 1:1 二进制兼容分支，这意味着您的应用程序和服务将保持兼容。因此，您只需一条命令即可快速迁移任意数量的服务器。

我们什么时候支持 AlmaLinux 操作系统？

AlmaLinux 合作伙伴已承诺在 2029 年之前支持 AlmaLinux 操作系统，包括稳定且经过全面测试的更新和安全补丁。
