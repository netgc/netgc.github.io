Capsa 便携式网络分析仪
监控、分析和排除有线和无线网络故障

Capsa 是一种 便携式网络性能分析 和诊断工具，提供功能强大且全面的 数据包捕获 和 分析 解决方案，界面易于使用，让老手和新手用户都能在关键业务环境中保护和监控网络 (sysin)。Capsa 帮助您评估可能导致重大业务中断的威胁。

Capsa 是适用于 LAN 和 WLAN 的 便携式网络分析器 应用程序，可执行实时数据包捕获功能、24x7 网络监控、高级协议分析、深度数据包解码和自动专家诊断。Capsa 对整个网络的综合高级窗口视图为网络管理员或网络工程师提供了快速洞察力，使他们能够快速查明并解决应用程序问题。Capsa 拥有业界最人性化的界面和最强大的数据包抓包分析引擎，是 网络监控 的必备工具。

Capsa Enterprise

Capsa Enterprise 专为小型企业和企业用户设计，可为有线和无线网络（包括 802.11a/b/g/n）执行网络监控、故障排除和分析。 无论您是拥有多年经验的老手还是网络监控的新手，如果您的需求是快速识别、诊断和解决网络问题，或者监控您网络上的用户活动，或者您只是需要确保公司的通信资产都是安全的，Capsa Enterprise 是必备的网络管理工具。

Capsa Enterprise 的主要特点

实时抓包
实时数据包捕获以及保存通过本地网络传输的数据的能力，包括有线网络和无线网络，如 802.11a/b/g/n。

高级协议分析
支持 1800 + 协议和子协议，包括 VoIP，以及基于协议分析的网络应用 (sysin)。

用户友好的仪表板
易于使用的概览仪表板让您一目了然地查看网络统计数据，从而快速解读网络利用率数据。

多种网络行为监控
可以监控和存储电子邮件和即时消息通信，帮助识别安全和数据处理违规行为。

快速查明网络问题
可以检测和诊断可疑主机，使您能够在几秒钟内查明网络问题。

每台主机的详细统计信息
映射网络上每台主机的流量、IP 地址和 MAC，以便轻松识别每台主机和通过每台主机的流量。

功能特性

  • 网络电话分析

    通过捕获和分析 VoIP 呼叫并显示结果，IT 人员可以对基于 VoIP 的网络进行基准测试和故障排除。

  • 通过电子邮件和音频通知警报

    当触发警报时 (sysin)，通过发送电子邮件或播放声音。

  • 任务调度程序

    它有助于在预定义的时间自动运行数据包捕获和分析，可以安排每天或每周运行一次。

  • TCP 流量分析

    TCP 数据包显示信息以解决慢速网络问题，如慢速网站响应、CRM 交易和下载等。

  • 网络安全分析

    准确检测 DoS（DDoS）攻击、Worm 活动、ARP 攻击、TCP 端口扫描和可疑会话，实时定位源和目标。

  • 多功能流量和带宽统计

    以图表和数字的形式监控网络流量和网络带宽详细信息。

  • 网络协议分析

    Capsa 能够支持 1800 多种协议，可以轻松分析网络中的协议并了解正在发生的事情。

  • 深度数据包解码

    它捕获网络上传输的所有网络数据包，并以十六进制、ASCII 和 EBCDIC 格式显示详细的数据包解码信息。

  • 多种网络行为监控

    监控 HTTP、电子邮件、DNS、FTP、MSN 和 Yahoo! 信使。

  • 每个主机的广泛统计

    列出网络中所有主机的详细信息（流量、IP、MAC 等）。

  • 自动专家网络诊断

    自动诊断网络问题并提出解决方案。

  • 矩阵中的可视化连接

    在椭圆中可视化整个网络，显示连接和流量。

  • 强大的对话分析

    监视所有对话并重建数据包流。

  • 有用且有价值的内置工具

    用于创建和重播数据包的免费内置工具；扫描并 ping IP。

  • 快速直观的报告

    快速生成最关心项目的报告。

系统要求

软件系统

操作系统：

  • Windows Server 2012/2012 R2/2016/2019（64 位）
  • Windows 7 SP1 (KB3033929)（64 位）
  • Windows 8.1（64 位）
  • Windows 10（64 位）
  • Windows 11（64 位）

浏览器：

  • Internet Explorer 8.0 或更高版本

硬件要求

最低要求：

  • 中央处理器：P4 2.8GHz
  • 内存：4GB
  • 浏览器 8.0

推荐系统：

  • 中央处理器：英特尔酷睿双核 3.2GHz
  • 内存：8GB 或更多
  • Internet Explorer 8.0 或更高版本

支持网络

  • Ethernet
    Capsa 将与 NDIS 3 或更高版本兼容的以太网、快速以太网或千兆混杂模式网络适配器一起运行。混杂模式是让 Capsa 接管驱动程序并将其置于一种模式，允许它被动地捕获以太网线上的所有数据包 (sysin)，而不管它们被发送 / 接收到的地址。
