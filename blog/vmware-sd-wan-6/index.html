产品概述

软件定义的 WAN (SD-WAN)

什么是 VeloCloud SD-WAN？

VeloCloud SD-WAN 是基于软件的网络技术应用，可虚拟化 WAN 连接。VeloCloud SD-WAN 将网络软件服务与底层硬件解耦 (sysin)，以创建虚拟化网络覆盖。VeloCloud SD-WAN 使用户能够随时随地连接到应用程序，提供灵活性、简单性、性能、安全性和云规模。它还具有易于部署、集中管理和控制以及有保证的应用程序性能。SD-WAN 是安全访问服务边缘 (SASE) 架构的重要组成部分。

降低带宽成本并将部署速度提高 10 倍

应对网络复杂性挑战，同时降低总体拥有成本并确保实现快速零接触式部署。

  • 性能和可靠性

    提供具有较高的性能、可靠性和传输能力的混合 WAN 以及提供商灵活性，以确保即使像语音和视频这样一些要求严苛的应用也能实现最佳性能。

  • 云网络

    利用云就绪网络消除数据中心回程开销 (sysin)，提供到公有和私有企业云的经优化的直接路径。

  • 虚拟服务平台

    单击一下即可减小分支机构的占用空间。借助 VMware SD-WAN，您可以在本地部署和云环境中无缝插入和链接虚拟化服务。

  • 自动化和编排

    集中式监控、可见性和云计算控制可实现零接触式分支机构部署，同时提供自动业务策略和固件更新、链路性能和容量测量。

功能特性

  • 支持迁移到云环境

    SD-WAN 提供混合、多云和 SaaS 功能，可满足当今企业需求。

  • 简化 WAN 运维管理

    通过自动化平台提供全局 WAN 可见性、敏捷性和扩展 (sysin)，从而使分支机构和远程站点部署变得简单。

  • 端到端虚拟服务平台

    SD-WAN 通过网络叠加将任何设备或任何用户连接到 WAN 上任何位置的任何应用，并提供深入的网络分析和安全保护。

  • 多种边缘安全机制选择

    云计算安全对 SD-WAN 至关重要，可借助 SD-WAN 平台，通过端到端分段和安全 VNF 实现。

解决方案组件

  • VMware SD-WAN Gateway™

    VMware SD-WAN™ 是一个由部署在全球各地顶层云计算数据中心的服务网关组成的分布式网络，提供了可扩展性、冗余和按需灵活性。VMware SD-WAN Gateway™ 提供通向所有应用 (sysin)、分支机构和数据中心的经优化的数据路径，还能从云端提供网络服务。

  • VMware SD-WAN Edge™

    VMware SD-WAN Edge™ 是一款零接触式企业级设备，能够以经过优化的方式安全地连接专有、公共或混合应用，以及计算和虚拟化服务。除了托管虚拟网络功能 (VNF) 服务外，VMware SD-WAN Edge™ 还执行深度应用识别、应用和数据包流量引导、性能计量，并维护端到端服务质量。

  • VMware SD-WAN Orchestrator™

    除了编排通过云网络的数据流之外，VMware SD-WAN Orchestrator™ 还提供集中式企业级安装、配置和实时监控。VMware SD-WAN Orchestrator™ 支持一键式置备分支机构、云环境或企业数据中心中的虚拟服务。

新增功能

New Features and Enhancements

  • 由 NSX™ 技术提供支持的增强型防火墙服务

    软件版本 6.0.0 中的增强型防火墙服务除了现有的入侵检测系统 / 入侵防御系统 (IDS/IPS) 之外，还具有 URL 过滤 和 恶意 IP 过滤 功能。

    URL 过滤 和 恶意 IP 过滤 服务由 VMware 屡获殊荣的 NSX 安全组件提供支持，使 IT 管理员能够减少系统的攻击面。通过 URL 过滤，网络流量根据类别和信誉进行过滤。将 NSX Security 的功能与 VMware SD-WAN Edge 平台相集成，使客户能够自信地删除分支机构的旧防火墙，而不会影响安全性，并体验简化网络和安全操作的优势。此外，客户还利用 VMware 在威胁情报方面的投资。

  • 安全服务组

    安全服务组是安全服务设置的有组织的集合，作为增强型防火墙服务的一部分提供。这些设置包括 URL 类别、URL 信誉、恶意 IP 和 IDS/IPS。这些组旨在通过跨多个防火墙规则创建和重用预定义安全服务配置来简化防火墙策略管理。这种方法无需为每个防火墙规则创建和维护多个单独的安全服务设置，从而简化了流程并提高了效率。

  • 增强的安全监控仪表板

    增强型安全监控仪表板提供专为企业量身定制的全面概述，通过 IDS/IPS、URL 过滤和恶意 IP 过滤提供对已识别威胁的深入洞察，从而增强企业的整体安全态势。

  • 改进的防火墙日志记录

    版本 6.0.0 中的防火墙日志记录功能为每个日志记录选择提供了一个全面的窗格视图，其中包含与防火墙和增强型防火墙服务引擎相关的数据。此外，还集成了新的智能过滤器，以便于基于特定引擎搜索日志，包括防火墙、入侵检测系统 / 入侵防御系统（IDS/IPS）、URL 类别、URL 信誉和恶意 IP 过滤。

  • Zscaler 自动化

    使用 Zscaler 云安全服务 (CSS) 自动化的客户现在可以在 Zscaler 位置名称字段中插入边缘名称，从而改善使用 Zscaler CSS 的大型企业的管理。

    备注：此增强功能仅适用于新的 CSS 配置。

  • 网关上的等价多路径 (ECMP)

    使用版本 6.0.0 的 SD-WAN 网关包括通过多个非 SD-WAN 目标 (NSD) 隧道进行等价多路径 (ECMP) 路由的功能。除了促进从网关到各种目标 IP 地址的大量主动 / 主动隧道外，ECMP 功能还允许流量均匀地分布在到达目的地的所有可用路径上，从而最大限度地提高带宽利用率并增强弹性。

  • Intel E810 NIC 支持网关部署

    使用版本 6.0.0 的 SD-WAN 和合作伙伴网关支持在 KVM 22.04 上使用 SR-IOV 的 100GbE 英特尔 ® 以太网网络适配器 E810，以实现高性能数据平面吞吐量。

  • Orchestrator GCP 支持

    版本 6.0.0 添加了对在 Google Cloud Platform (GCP) 上托管 Orchestrator 的支持。
