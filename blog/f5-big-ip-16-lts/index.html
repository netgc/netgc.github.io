BIG-IP 16.1.0 是 BIG-IP 系统的最新长期维护版本（LTS），持续的缺陷和漏洞修复将添加到维护版本中。BIG-IP 16.1.0 中发布的新功能包括增强企业和服务提供商的安全性以及改进和简化的用户和管理体验。

从 BIG-IP 16.1.0 的发布开始，对于后续版本，F5 的长期稳定性版本标准支持生命周期从五年变为四年。

笔者注意到，随着 16.1.0 “LTS” 词汇的首次使用，之前的版本 x.1.0 也全部称为 LTS 版本。

BIG-IP 16.1.0

使用 BIG-IP 16.1.0：

  • 企业客户可以：

    • 增强零信任架构 (ZTA)（BIG-IP APM / Microsoft Azure Active Directory 条件访问集成）
    • 确保用户的登录凭据安全且未被泄露（泄漏凭据检查）
    • 减少用户登录摩擦并增强用户体验，同时提高登录安全性并减轻撞库等攻击（基于 iApp 的 F5 Device ID+ 与 BIG-IP 的集成）
    • 应对高度复杂的机器人攻击并消除欺诈（基于 iApp 的集成机器人防御*与 BIG-IP 的集成）
    • 增强 API 和 Web 应用程序安全性（通过 APM 访问引导配置进行 API 访问和安全性 (sysin)；提高 GraphQL 的 API 安全性；通过机器学习提高安全性；缓解 SSRF）
    • 安全地解决 Web 协议的增强（HTTP/2 支持）

  • 服务提供商可以使用 DNS over HTTPS (DoH) 解析加密的 DNS 查询并缓解 DNS 欺骗攻击

  • 用户和管理员还可以享受增强的体验（使用 F5 Edge 客户端进行动态拆分隧道；通过 HTTP 批处理请求提高效率、性能和可用性；以及作为 SSL Orchestrator 服务的 F5 SWG 服务）

*请注意，集成机器人防御目前的可用性有限/预览。
