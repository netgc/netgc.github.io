Cisco ASR 9000 系列聚合服务路由器

专为满足苛刻的边缘市场而打造的高性能路由器

边缘网络为数十亿台设备提供低延迟服务。您需要功能丰富的路由器，可以根据带宽需求经济地扩展。ASR 9000 系列是您的边缘路由设备，能够支持满足 5G 服务需求所需的应用性能。

为今天而打造，为接下来的一切做好准备

重装上阵

  • 增强的端口灵活性，支持 1G 到 400G 和新的外形规格。
  • 支持相干光学以推动 IP 和光学层的融合。
  • 致力于向前和向后兼容 (sysin)。

重构

  • 使用简单、现代且值得信赖的网络操作系统 (IOS XR)。
  • 使用开放 API、本机、YANG 和开放配置模型实现自动化。
  • 具有实时遥测、分段路由和以太网虚拟专用网络 (EVPN)。

有弹性的

  • 增强的安全性，包括安全启动、Cisco Trust Anchor 和防伪措施。
  • 通过安全的开发生命周期来确保制造的完整性 (sysin)。
  • 通过持续的软件和硬件创新为 ASR 9000 提供投资保护。

可持续性

  • 定制 ASIC 经过优化，可在性能与 100 倍容量增加和每 Gbps 功耗降低 96% 之间取得平衡。

优势

满足大规模网络的需求

ASR 9000 是一种高密度、高性能的边缘平台，适用于不断增长或高需求的市场。ASR 能够支持从 1G 到 400G 的连接，为运营商提供一个超前扩展的长期增长平台。

保护和优化您的投资

今天就在一个准备好为明天扩展的平台上构建您的网络。借助灵活的按需付费选项和动态许可证池，您可以简化容量规划和扩展以满足需求。

加速服务敏捷

使用 Cisco IOS XR 的多维可编程性、开放式 API 和高级遥测，提供前所未有的网络可见性和由分析和机器学习驱动的闭环修复。随着您的网络基础设施的发展，IOS XR 和 Cisco 自动化工具为您的工程师提供精细控制，以便他们能够快速采取行动为您的客户提供最佳客户端体验。

从内到外保护您的关键基础设施

验证在您的网络中运行的 ASR 硬件和软件的完整性，以促进网络堆栈每一层的保护。通过这些验证，您可以确保您的关键基础设施不变，然后通过自动软件升级来改善您的安全状况，确保您的网络安全和最新。
