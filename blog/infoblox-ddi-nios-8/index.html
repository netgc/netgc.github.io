什么是 NIOS

Network Identity Operating System (NIOS) 是为 Infoblox 核心网络服务提供支持的操作系统，可确保网络基础设施的不间断运行。作为 Next Level Networking 的基础，NIOS 可自动执行与部署和管理 DNS、DHCP 和 IP 地址管理 (IPAM) 相关的容易出错且耗时的手动任务，以实现持续网络可用性和业务正常运行时间。

NIOS 操作系统是一个强化的 Linux 发行版。
