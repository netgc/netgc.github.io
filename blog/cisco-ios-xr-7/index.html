思科 IOS XR7 网络操作系统

​

简单、现代化、值得信赖的网络操作系统

全新的现代化思科 IOS XR7 具备更优的模块化设计、简化的网络堆栈和更强的云自动化功能 (sysin)，可提高网络可编程性，从而降低运营复杂性。

  • 查看产品手册

特性和优势

​

简单

IOS XR7 采用灵活的模块化设计，使用的内存更少，启动速度更快，还可加载到核心层或接入层设备中 (sysin)，因此能够帮助简化您的网络。在整个网络中使用统一的操作系统 (OS) 将简化工程师的操作，让他们腾出精力专注于创收任务。

现代化

借助现代化网络操作系统让您的工程师重新获得卓越的可控性和强大的功能。集成的数据流遥测数据可提高网络可视性，并允许工程师针对网络事件使用开放式 API 制定主动应对策略，让客户满意。工程师还可以利用这些数据建立更出色的容量耗尽模型，对资本支出和运营支出预算加大控制。

值得信赖

IOS XR7 是有助于提高网络安全性的高级网络操作系统。只有在完成防篡改自检过程后，它才允许 CPU 启动 (sysin)。此外，它还提供了有效的保护措施，以防硬件和固件受到损害。IOS XR7 通过高级签名技术和包括集成测量架构 (IMA) 在内的多种运行时防御功能来防御恶意攻击者和漏洞攻击所利用的漏洞。

IOS XR 网络操作系统支持的平台

凭借辉煌的历史、一流的路由协议和更出众的网络可控性，IOS XR 适用于从边缘到核心的各种路由平台。
