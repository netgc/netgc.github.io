    将 PDF 文件中的图像和形状元素编辑为图层，并在导出时保留所有矢量数据、混合模式、不透明度和蒙版。

    打开和编辑创建的 Safari、Keynote、Pages 文稿和其他 App 的 PDF 文件。

  • 使用快捷方式转换为 PDF

    使用 “快捷指令” 应用将多个 Pixelmator Pro 文稿和其他文件转换为 PDF。

Pixelmator Pro 3.2 引入了一些令人难以置信的新方法来创建完全支持视频层的设计，添加了对流行视频格式的支持，包括全新的视频和社交媒体模板，带来了对 Motion 文档支持的重大改进，等等。

  • Motion 支持改进

    享受对 Motion 文件支持的许多改进，包括视频层、非破坏性颜色调整和效果等。

功能概览

了解 Pixelmator Pro 的强大功能。

  • 用于完善照片的强大工具

    Pixelmator Pro 中强大的非破坏性颜色调整集合可让您以任何想要的方式编辑照片中的颜色。凭借对 RAW 照片的全面支持、一系列令人惊叹的调整预设和令人难以置信的修饰工具，将好看的照片变得壮观再简单不过了。

  • 使用完整的矢量工具集进行绘图和说明

    Pixelmator Pro 还附带一整套矢量工具，用于创建与分辨率无关的设计。它包括易于定制的智能形状、大量预先设计的形状，并支持 SVG 矢量文件格式。

  • 创造令人惊叹的设计所需的一切

    无论您是在设计海报、网页还是轰动一时的应用程序，您都可以在 Pixelmator Pro 中将您的想法变为现实。借助智能间距参考线、高级对齐工具和极快的性能 (sysin)，您可以充分利用基于图层的编辑功能。

  • 轻松创作精美的画作

    Pixelmator Pro 包括一系列令人惊叹的手工制作的双纹理画笔、完整的图形输入板支持和先进的绘画混合技术。由 Apple 开发的先进图形加速技术 Metal 驱动的尖端绘画引擎可提供令人难以置信的性能。

  • 选择适合您的工作空间

    自定义 Pixelmator Pro 工作区以使其适合您——将工具和图层侧边栏移动到您想要的任何位置，并完全自定义工具列表。或者选择专为摄影师、设计师、画家和插画家创建的内置预设之一。

  • 由机器学习提供支持的图像编辑

    为了提供更智能的图像编辑，Pixelmator Pro 使用了机器学习 —— 一种允许计算机获得知识以执行特定任务的技术，而不是像计算机一样。这使我们能够创建前所未有的功能。

  • 神奇地摆脱不需要的物体

    通过使用修复工具在图像上绘画，使小的瑕疵甚至整个对象从图像中消失。无论背景多么复杂，修复工具几乎可以让任何物体消失得无影无踪。

  • 逐张编辑您的照片，或将多张照片合二为一

    Pixelmator Pro 拥有一系列由最先进技术提供支持的像素精确选择工具，可让您轻松挑选和编辑图像的精确部分。因此，您可以将颜色调整和效果应用于特定区域。选择对象并将对象从一个图像复制到另一个图像。或者将所有编辑集中在一个精确的区域而不影响图像的其余部分。

  • 轻松设计

    使用 Pixelmator Pro 中强大的文字工具，您可以轻松地在所有图像中设计出漂亮的文字。沿着形状和路径轻松创建弯曲的文本或文字。并使用一整套排版工具以您可以想象的任何方式自定义文本。

  • 实时、无损、轰动一时的效果

    Pixelmator Pro 带有超过 60 种多功能效果的集合，您可以混合和组合以开发您可以想象的任何艺术或特殊效果。效果是非破坏性的，因此您始终可以在应用效果后对其进行编辑、重新排列或删除。您甚至可以将多种效果应用于单个图层 - 实时查看所有更改。

  • 将您的照片应用程序变成图像编辑的强者

    Pixelmator Pro 编辑扩展使整个 Pixelmator Pro 应用程序及其所有功能都可以在照片中使用。因此，您可以使用 Pixelmator Pro 进行编辑而无需离开照片，将所有非破坏性编辑保存在您的库中。您的照片应用程序现在是功能齐全的基于图层的图像编辑器。

  • 高级工作流程的高级功能

    Pixelmator Pro 从头开始设计，易于初学者使用。但随着您开始掌握图像编辑，它已准备好与您一起进步，因此您还会发现许多高级功能，如导出为 Web、软校样、对 Adobe Photoshop 文件的支持、色彩管理、AppleScript 等。

  • 专为 Mac 设计和构建

    从它的应用程序图标、外观和感觉到它支持的技术，Pixelmator Pro 被设计为终极 Mac 应用程序。享受对您期望的每项 Mac 功能和技术的全面支持 — iCloud、版本、快捷方式、Sidecar、Apple Pencil 等等。

由最新的 Mac 技术驱动

Pixelmator Pro 图像编辑引擎非常复杂且功能强大。它专为利用 Mac 电脑的全部功能而设计，使用了 Metal 和 Core Image 等先进的 Mac 图形技术。Pixelmator Pro 突破了图像编辑的界限，还包括由 Core ML 提供支持的机器学习增强型编辑功能。因此，工具不仅快速起泡——它们也非常智能。

  • Apple M 芯片

    Pixelmator Pro 可在搭载 Apple M 芯片的 Mac 上原生运行，充分利用其令人难以置信的性能。

  • Metal

    使用 Metal，Pixelmator Pro 可以充分利用每台 Mac 的图形处理能力。

  • Core ML

    Pixelmator Pro 中突破性的机器学习功能使用 Core ML 进行集成，这在 Mac 上带来了最佳的 ML 处理性能。

  • 使用 Swift 构建

    Swift 是一种现代编程语言，旨在提高效率、可靠性和一流的性能。

  • Core Image

    Core Image 大大加快了图像处理速度，实现了快速、无损的编辑。

  • SwiftUI

    高性能 UI 动画由 SwiftUI 提供支持。
