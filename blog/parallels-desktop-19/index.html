Parallels Desktop 19 for Mac 发布, 简化 macOS 和 Windows 交互

August 21, 2023

Mac 爱好者大家好，

这是多么令人难以置信的一年啊！

我们非常高兴地宣布 Parallels Desktop 19 for Mac 的下一轮创新。

Parallels Desktop 已成为生产力和软件开发的催化剂，使 Mac 用户能够毫不妥协地利用多个操作系统的优势。

超过 200 万专业人士、开发人员、创意人员和爱好者信赖 Parallels Desktop 在虚拟环境中运行 Windows、Linux 和 macOS。

可能性是无限的，对行业、企业和个人努力的影响怎么强调也不为过。

这篇博文将探讨此版本中的新增功能以及不同的用户群体将如何从中受益。

与 macOS Sonoma 升级的兼容性

我们知道在持续软件升级后保持正常运行是多么重要 (sysin)。这就是为什么与 macOS 的兼容性始终是并将继续成为主要关注点的原因。

在 macOS Sonoma 14 中，转换 PostScript 文件的软件模块 CGPSConverter 已从 macOS 中删除。结果，共享打印功能变得毫无用处，因为它使用此模块允许通过 Mac 共享的打印机从 Windows 进行打印。

对于许多 Parallels Desktop 用户来说，从 Windows 进行打印至关重要。作为回应，我们的工程师重新设计了共享打印功能，以使用当今最流行的打印方法互联网打印协议（IPP）。IPP 打印具有向 Windows 公开更多打印机特定功能的潜力，我们的工程师目前正在探索这一点。

在虚拟机中运行 macOS Sonoma 也有一些改进，我们将在下面的单独主题中介绍。

设计

作为一款 Mac 应用程序，我们深受 Apple 设计细节深度的启发。

我们花费了时间和精力来满足现代 macOS 设计准则。

我们不想简单地将旧图标放在熟悉的圆角白色瓷砖上 (sysin)，而是重新思考如何在 macOS 中以视觉方式传播我们的应用程序。

新的应用程序图标和产品内详细信息实现了这一点！

  • 鉴于我们大多数用户都拥有 MacBook 电脑，我们将长期使用的类似 iMac 形式的背景更改为笔记本电脑。
  • 同时运行多个操作系统的并行隐喻已经从 Mac 前面转移到后台，因为这就是我们所处的位置 - 启动虚拟化引擎并在后台运行，使用户能够运行第三方操作系统和软件。
  • 最后，左上角的神秘漏洞已得到解决。角上有一个点的矩形代表一个容器：一个托管“您的东西”的窗口，无论您从哪个平台访问它。此外，这是对新 Parallels 品牌的认可，该品牌将所有 Parallels 产品整合到一个供组织使用的钥匙串中。

与 Mac 集成：使用 Touch ID 进行 Windows 登录

我们将 Apple Touch ID 的魔力带入 Windows 登录和 Parallels Desktop 安装！

如果您的 Windows 10 或 11 受 PIN 或密码保护，则升级到 Parallels Desktop 19 后下次登录 Windows 时将提示 Windows 使用提供的新 Parallels 凭据。然后，您的 Windows 密码将使用特殊的 Touch ID 绑定记录类型（仅可用于 Touch ID 使用的记录）保存到 Mac 钥匙串中 (sysin)，仅在使用您的指纹的这台 Mac 上可用。

下次您登录 Windows 时，Parallels Desktop 将提示您使用 Touch ID 登录您的用户帐户。它既简单又安全。用户可以通过简单但有用的界面管理保存的密码，并根据需要恢复使用旧的登录方法。

对于某些安全策略要求使用密码保护 Windows 的组织中的 Parallels Desktop for Mac Business Edition 用户来说，此改进将非常方便。

Mac 上的专业 Windows 软件：OpenGL 改进等

我们感到非常兴奋的是，世界各地的建筑、工程和设计专业人士都使用 Parallels Desktop 通过工作所需的 Windows 应用程序为他们的 Mac 提供支持。今年，随着 OpenGL 支持改进至版本 4.1，我们允许更多仅适用于 Windows 的 CAD 软件在 Mac 上运行，包括 VariCAD、Deswik.CAD、Vectorworks Vision 2023 等。

对于信任 Parallels Desktop 在 Mac 上运行流行的地理信息系统软件 AcrGIS Pro 的专业人士来说，还有个好消息。我们听取了反馈，并很高兴地宣布此版本中 ArcGIS Pro 的性能得到显着改进。

Apple 芯片上的 macOS VM：更容易安装、更多操作和控制

在采用 Apple 芯片的 Mac 上，macOS 虚拟机由相对较新的 macOS 框架提供支持，与用于启动 Windows 和 Linux（在所有 Mac 计算机上）的 Hypervisor 框架相比 (sysin)，Parallels Desktop 对虚拟机操作的控制要少得多，以及基于 Intel 的 Mac 上的 macOS。这就是为什么 Apple 芯片上的 macOS VM 功能少得多，且捆绑格式“.macvm”与用于其余 VM 类型的熟悉的“.pvm”格式不同。

此升级包含针对 Apple 芯片上的 macOS 虚拟机的改进：

  • VM 安装：使用 IPSW 映像通过安装助手 UI 或 CLI ($ prlctl create “macOS13” -o macos -restore-image “/path/to/image.ipsw”) 安装新的 macOS 虚拟机。
  • 基本 VM 操作：从 UI 菜单或使用 Parallels CLI prlctl 重新启动、正常关闭以及暂停/恢复虚拟机。
  • 输入：在虚拟机中使用两指触控板手势进行滚动和缩放。
  • 大规模部署：将带有 macOS 虚拟机的 Parallels Desktop 部署到带有 Apple 芯片的 Mac 计算机。

当 macOS Sonoma 14 在主容量和虚拟容量中运行时，体验会更好：

  • 显示：调整虚拟机窗口大小时，显示分辨率会自动更新。
  • 网络：在专业版中，使用端口转发通过 SSH 或 RDP 远程访问 macOS 虚拟机，例如，当您将它们托管在 Amazon EC2 Mac 实例上时。

将 Packer 与 Parallels VM 结合使用更容易，并且支持 Apple 芯片上的 macOS VM

当我们开发 Parallels Desktop 时，我们总是尝试尽可能地自动化体力劳动。这同样适用于我们许多使用 Parallels Desktop 开发和测试软件的客户。为了响应客户的宝贵反馈，我们与 HashiCorp 合作，为 Packer 和 Vagrant 的 Parallels Provider 提供期待已久的改进。

借助 Parallels Desktop 19，我们不再需要使用 Parallels SDK 和 Python，从而可以更轻松地使用这些基本工具。我们很高兴地宣布，此次升级还带来了在 Apple Silicon 上创建和配置 macOS 虚拟机的能力。

作为我们对开源承诺的一部分 (sysin)，我们为 Packer 开发了各种示例模板，简化了新客户的入职流程。这些模板适用于 Windows、Linux 和 MacOS，并且可以在我们的 GitHub 存储库上免费获取。

此外，我们还修改了文档，提供了 全面指南。与我们的提供商无缝使用这些工具的

Apple 芯片上的 Linux：运行 x86 Docker 容器和 Microsoft SQL、CentOS 9 Stream

对于需要在采用 Apple 芯片的 Mac 上运行 x86 Docker 容器或 Microsoft SQL Server 的开发人员，Parallels Desktop for Mac Pro Edition 提供了一个新的即用型 Ubuntu Linux Arm VM，该 VM 配置为运行 x86 Docker 容器，由 Apple 提供支持罗塞塔2技术。新的 VM 可以从安装助手的免费系统部分下载。

而且，对于开发人员和高级用户来说，这是一项改变游戏规则的举措，我们很高兴推出在采用 Apple 芯片的 Mac 上的 Linux Arm VM 中运行 x86 二进制文件和容器的功能。通过整合现成的可下载 Ubuntu Linux VM，我们简化了运行 x86 容器映像（包括 Microsoft SQL Server 等流行容器映像）的过程。

这一突破使用户能够轻松使用和调试容器，并为 x86 平台构建它们。

展望未来，我们致力于在未来的更新中不断提高这些二进制文件的兼容性。请继续关注该领域令人兴奋的发展。

Parallels Desktop 19 for Mac 还支持在采用 Apple 芯片的 Mac 上运行 CentOS 9 Stream。无论您是学习、开发还是测试 CentOS/RHEL，我们的解决方案都能提供灵活性和多功能性。

组织中的设备管理：将 Windows 注册到 Microsoft Intune

我们了解设备管理对于组织的重要性，并且 我们的客户越来越多地采用 Microsoft Intune。

以前，为了确保组织中最终用户使用的虚拟 Windows 设备可以注册到 Microsoft Intune，IT 管理员只能使用大规模部署方案，这对于某些类型的组织来说并不方便。

以及通过云存储共享 Windows 映像时在 Intune 中注册 Windows 11。配置 Parallels Desktop 19 for Mac Business Edition 能够在通过 Parallels My Account Web 应用程序向员工

此功能使管理组织内的 Windows 设备变得更加容易和高效。

Parallels Desktop 旨在弥合 macOS 和您选择的操作系统之间的差距，使您能够毫不妥协地享受两个世界的最佳效果。

新版本的 Parallels Desktop 为希望运行 Windows 或 Linux 的 Mac 用户提供了更加无缝的体验。

我们很自豪能够成为 Microsoft 授权的解决方案 (sysin)，我们的开发证明了我们致力于提供高质量、强大、可靠且高效的解决方案以在 Mac 上运行 Windows 11。

我们渴望您体验与 Parallels Desktop 的更高水平的 Windows-macOS 集成。
