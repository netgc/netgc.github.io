出色防御基于互联网的威胁

高级的威胁甚至可以隐藏在合法网站上。用户可能会无意中点击不应该点击的地方，从而使您的组织面临风险。通过自动拦截危险网站以及在允许用户点击未知网站之前检测该网站 (sysin)，Cisco Secure Web Appliance 成功保护您的组织。使用 TLS 1.3 和高性能功能，Cisco Secure Web Appliance 成功确保您的用户安全。

​

特色

强有力的保护

Cisco Secure Web Appliance 提供了多种方式来自动检测并拦截基于网络的威胁。Cisco Secure Web Appliance 高级许可解决方案由我们的 Talos 威胁研究组织提供支持，提供深入的 URL 过滤和信誉分析、多个防病毒引擎、第 4 层流量监控 (sysin)、面向 Secure Web Appliance 的恶意软件防御以及感知威胁分析 (CTA)。

精准控制

Cisco Secure Web Appliance 的作用远不止于拦截或允许特定网站，它还可以让管理员禁止或允许站点内的特定功能。除此之外，管理员还可以为某些用户破例，并为网络的使用设置时间和带宽限制。这样，组织便能够阻止有关行为，而不会影响最终用户的工作效率。

详细报告

借助 Cisco Secure Web Appliance，安全团队可以轻松查看正在访问特定网站的用户以及其访问时间。详细的报告选项让管理员可以通过以下方式显示数据：

  • Cisco Secure Web Appliance
  • Cisco Secure Email 和 Web Manager（原“安全管理设备”），适合集中管理
  • 思科高级网络安全报告，可通过 Cisco Umbrella 自定义报告和混合报告

灵活的部署选项

思科提供各种部署选项，以便 Cisco Secure Web Appliance 随着组织需求的增长而扩展并发展。您可以使用 Amazon Web 服务将其部署为硬件设备或虚拟机，也可以将其部署在公共云中。与思科携手，获得与时俱进的保护。

集成的安全架构

借助思科的集成安全产品组合，您可以将 Cisco Secure Web Appliance 与其他技术结合使用，从而获得更全面的保护。与 SecureX 的集成为整个 Cisco Secure 产品套件提供了更好的可视性和自动化性能，助力您的企业安全运作。

Cisco Secure Web Appliance：虚拟和云端

Cisco Secure Web Appliance 还可以通过虚拟外观使用 (sysin)，也可以通过 Amazon Web 服务在公共云中使用。Cisco Secure Web Appliance 虚拟机是 Cisco Secure Web Appliance 的软件版，在 VMware ESXi、KVM 虚拟机管理程序、Microsoft Hyper-V 和思科统一计算系统 (Cisco UCS) 服务器上运行。
