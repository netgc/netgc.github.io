Magnet AXIOM - 恢复并分析一个案件中的证据

检查来自移动、云、计算机和车辆来源的数字证据，以及第三方提取的数据，所有这些都在一个案例文件中。使用强大且直观的分析工具快速自动显示案件相关证据。

新工具如何消除干扰寻找证据

调查中涉及的数字设备数量不断增加，平均每人大约有 6 台设备*，这使得采集、处理和分析在逻辑上充满挑战、耗时且昂贵。AXIOM 等工具使审查人员能够简化工作流程并消除数字干扰 (sysin)，从而更快地定位、恢复和收集证据。

AXIOM 功能

使用 Magnet AXIOM 在一个案例文件中恢复、分析和报告来自移动、计算机、云和车辆来源的数据。

  • 强大的工件复原
  • 先进的分析工具
  • 移动工作流程
  • 增强您的证据工作流程

强大的工件复原

采用工件优先的方法，轻松恢复已删除的数据并分析一个案例文件中来自移动、计算机、云和车辆来源的数字证据。发现文件或工件的完整历史记录以构建您的案例并证明意图 (sysin)。Magnet AXIOM 为最新的设备和源提供最新的工件支持。

要点：

  1. 在一个案例中同时获取和分析移动、云和计算机证据。
  2. 处理来自 Google、Facebook 和 Instagram 等提供商的保证金退货。
  3. 检查来自 Google、WhatsApp 等云源的开源数据和用户帐户数据。
  4. 处理图像——从提取、数据恢复到案例文件构建——全部一步完成。

先进的分析工具

使用 Magnet AXIOM 的分析工具自动显示更多证据，以便您可以专注于与您的案例相关的内容。利用 Media Explorer、Cloud Insights Dashboard、Magnet.AI、Connections、Timeline 等快速找到您需要的证据。

要点：

  1. 使用 Magnet.AI 和 Thorn 等机器学习工具自动检测潜在的非法内容图片，例如虐待儿童、毒品和武器。
  2. 使用 “连接” 可以快速拼凑工件、人员甚至设备之间的相互关系。
  3. 使用 Media Explorer快速显示图像和视频中的智能见解。
  4. 使用时间线可视化所有证据来源的事件 (sysin)。
  5. 按日期和时间范围、特定工件或感兴趣的项目以及关键字过滤数据，以快速找到相关证据。

移动工作流程

无论您选择哪种采集工具，Magnet AXIOM 都能获取最多的数据并为 iOS 和 Android 设备提供最佳的分析。通过将 GrayKey 直接集成到 AXIOM 中 (sysin)，可以轻松加载移动证据以进行深入分析。

要点：

  1. 通过直接 GrayKey 集成以及对 Cellebrite、Oxygen、Berla 等第三方工具的支持，摄取和处理移动设备提取。
  2. 使用 AXIOM 中强大的雕刻功能来发现图片、聊天和浏览器历史记录。
  3. 利用 KnowledgeC、Android Motion Photos、iOS Wallet、Samsung myFiles、地理位置数据等工件揭示详细的主题信息。
  4. 通过利用移动设备中的令牌和钥匙串来执行自动解密。

增强您的证据工作流程

Magnet AXIOM 旨在与其他 Magnet 取证工具集成，帮助您转变数字调查，使您、您的实验室和您的机构能够满足不断增长的容量需求并在机构范围内进行协作，同时安全、透明地操作以降低风险。

要点：

  1. 使用 Magnet AUTOMATE 创建标准化工作流程，以在检查员干预有限的情况下对数据进行成像和处理。
  2. 与非技术利益相关者安全地共享数字证据，他们可以使用 Magnet REVIEW 从任何地方审查证据。
  3. 从 Magnet OUTRIDER 保存的文件中提取并自动显示工件 (sysin)。
  4. 使用 Magnet ATLAS 生成实时报告并跟踪数字证据，确保维护监管链。
