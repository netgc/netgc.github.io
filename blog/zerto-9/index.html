Zerto 9.5 U1 过渡到 Linux 以减少攻击面

作者主页：sysin.org

最新版本的 Zerto 9.5 U1 开始向更安全的 Linux 设备过渡。在第一个版本中过渡尚未完成 (sysin)，因此已经在使用 Zerto 的客户还不能迁移到 Linux。从 Windows 迁移将在即将发布的版本中提供。

这个新版本还为客户门户网站、MyZerto 和 Zerto Analytics 带来了新的基于角色的访问控制 (RBAC) 和多因素身份验证 (MFA)，以及复制设备更新和对组件之间 SSH 通信的更精细控制。

Zerto 的复制和备份解决方案与其他备份供应商有很大不同。它 不使用管理程序的快照 进行备份或复制。

Zerto 的设备 (VRA) 能够利用虚拟机的 IO 流，与传统的管理程序快照相比，不会导致开销或性能问题。它从特定 VM 捕获 IO 并将其复制到远程站点，日志系统将这些数据排序最多 30 天。

Zerto 使用 Elastic Journal 允许您短期和长期保留您的虚拟机。弹性日志是一个连续的恢复点流，可从 7 秒到多年的任何时间点跨数据、文件或虚拟机进行搜索和恢复。

Zerto 9.5 U1 有什么新功能？

Zerto 是一款功能强大的 DR 解决方案，易于设置、测试和使用 (sysin)。您还可以使用 Zerto 在数据中心和云提供商之间迁移我们的虚拟机。

  • 数据保护——本地持续备份和长期保留

    增强的文件级还原体验 ——您现在可以选择您的来源（期刊或存储库），选择一个时间点和要还原的虚拟机，然后浏览要还原的文件和文件夹，所有这些都来自一个统一的简化还原向导。

    适用于 Linux 的即时文件还原 - 您现在可以将 Linux 文件执行文件级还原到运行 Linux 的生产 VM。以前仅支持 Windows 文件和 VM 将文件还原到生产 VM。

    从复制中排除磁盘 - 现在可以通过 Zerto 连续数据保护将虚拟机磁盘卷从复制中排除。

  • 长期保留 - 存储库

    可以通过 REST API 创建和管理长期保留存储库。

    新的不变性选项 ——Microsoft Azure Blob 存储现在是用户直接从 Zerto 界面创建和管理不可变数据副本的目的地，以增加对修改或删除的保护。

    Zerto 正在扩展对第三方 S3 兼容存储的不变性的支持和管理。

    Scratch Journal 设置 - 由于 Journal 和 Scratch Journal 用于不同的目的并且有不同的要求，因此它们现在具有独立的设置。您现在可以选择专用数据存储并为暂存日志设置硬限制。

    Zerto 登录横幅 - 引入了自定义登录横幅，为组织提供可自定义的安全通知，以便在访问受限系统时提供指导。

    恢复 VRA 弹性 ——提高了在包含 VRA 的 ESXi 主机上发生硬件故障时的弹性，可以重新配置 Zerto 以继续在集群中幸存的 ESXi 主机上执行复制和数据保护任务。

    Linux 网络管理器对 Re-IP 的 支持——为使用 bash 和网络管理器的较新版本的 Linux 添加了对 Re-IP 的支持，包括 Ubuntu、RedHat、CentOS、Debian、Suse 和 Oracle。

    虚拟复制设备 ——在 Debian 9 上运行的虚拟复制设备已升级到 Debian 11，其中包括安全合规性所需的安全补丁。

Linux 设备常见问题解答

由于 Linux 设备非常新，这里是 Zerto 网站的常见问题解答。

  • 它是什么/正在发射什么？

    Zerto 正在启动一个 Zerto 虚拟管理器，它在安全的 Linux 操作系统而不是 Windows 上运行。

  • 我为什么在乎？

    Linux ZVM 允许客户最小化与 Zerto 相关的攻击面。该设备不运行其他服务，只需要公开 Zerto Web 界面。

  • 我该如何开始？

    如果您是 Zerto 的新客户，您只需下载 OVF 模板文件并通过 VMware 进行部署。对于现有客户，从 Windows 迁移到 Linux 上的 ZVM 将在即将发布的版本中提供。

  • 我是否需要熟悉 Linux 才能使用 Linux ZVM？

    Linux ZVM 不需要比 Linux vCenter Server Appliance 更多的工作。如果您熟悉 vCenter，您将能够管理 Linux ZVM。所有日常 Zerto 管理功能都通过 Web 界面实现。

  • 是否支持 Hyper-V？

    Linux ZVM 仅支持 VMware vSphere 7.0 和更新的环境。

  • 我可以将 Linux ZVM 与 Windows ZVM 或 Cloud ZCA 配对吗？

    目前不支持异构配对，但将在 Zerto 的未来版本中添加。Linux ZVM 9.5 U1 版本仅支持 Linux ZVM 到 Linux ZVM 站点配对。

  • Linux ZVM 是否具有与 Windows ZVM 相同的功能？

    Linux ZVM 9.5 U1 版本具有部分功能。请参阅 Linux 上的 ZVM：要求、支持的功能和配置。
