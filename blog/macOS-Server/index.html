摇身一变，你的 Mac 就是 一台强大的服务器

macOS Server 为企业、家庭办公室和学校带来更多动力。macOS Server 为配合 macOS 和 iOS 设备使用而设计，让你能轻松配置 Mac 和 iOS 设备。而且它的安装、设置和管理也非常轻松简单。

你的指挥部

有了 macOS Server，你可以直接在 Mac 上设置和管理多部 Mac 电脑和 iOS 设备。而且它非常简单易用，没有 IT 部门也没问题。

提升管理水平，秘诀就这么简单

macOS Server 的先进功能让你能轻松控制企业的硬件和软件。因此，你可以更高效地设置和管理每一部设备。

描述文件管理器 多任务处理大师

有了描述文件管理器，在你的组织中部署、配置和管理 Mac 电脑及 iOS 设备都变得更简易。你可以在这里控制一切：通过创建描述文件来设置邮件、日历、通讯录和信息的用户帐户，配置系统设置，实行限制，设置 PIN 和密码规则等。凭借描述文件管理器，分发通过 App Store 批量购买计划所购得的机构授权 app 比以往更为简便。它还能让用户访问带有自助服务的门户网站，并从那里下载和安装新的配置描述文件，还可清除密码，远程锁定或抹掉丢失或被盗的 Mac、iPhone 或 iPad。它能让你进行 macOS 企业 app 的推送安装，也支持 PDF、EPUB 及 iBooks Author 文件等 iOS 媒体素材的推送安装。

适用于 iOS 和 macOS 的功能

macOS Server 可让你将通过批量购买计划 (VPP) 获得的各种 app 分配至每台设备，无需使用用户的 Apple ID。这样即可在 iOS 设备和 Mac 电脑上安装通过 VPP 获取的各种 app，不用再配置 Apple ID 或发送邀请。你还可以将设备上已安装的 app 迁移至用户的 Apple ID 中，而无需删除该 app 或用户数据。

适用于 iOS 的功能

即使 App Store 暂不可用，仍可安装 App Store 中的 app，包括新分配的 app 及 app 更新。对于使用 Kerberos 协议的 app，为该 app 单独配置的 VPN 会在用户登录时自动开启。你可以将 iPad 和 iPhone 加入设备注册计划 (DEP)，也可移除 “从 Android 设备转移” 选项。网络使用规则允许每个机构或公司指定受管 app 使用网络的方式，例如限制 app 通过蜂窝网络连接或者在其他网络中漫游。你还可以将参加 DEP 的设备更新至 iOS 的最新版本。

macOS Server 的设备限制功能会阻止使用 Mail Drop 或 AirDrop。你也可以启用对受监管设备的限制，比如，禁止更换墙纸，禁止更改设备名称，禁止修改企业 app 信任设置，禁止访问 iCloud 照片或键盘快捷键，禁止与 Apple Watch 配对，禁止设置密码。

适用于 macOS 的功能

初次设置系统时，管理员帐户会自动创建，你可以隐藏此帐户，使普通帐户无法看到。你也可以在 DEP 注册过程中创建普通帐户或跳过帐户设置步骤，只需配置 macOS 设置助理，即可在 DEP 注册过程中创建新的普通帐户 (非管理员)，或跳过帐户创建步骤。

Xsan 科技锐进，大大超前

Xsan 是一款强大的、可扩充的存储整合解决方案。你机构中的每个人都可以快速地同时访问 TB 级的集中数据。Xsan 内置于 macOS 之中，可让任何 Mac 通过光纤通道或以太网访问 Xsan 或 StorNext 卷宗。

基本要求

  • 运行 macOS Mojave 的 Mac 电脑
  • 2GB 内存
  • 10GB 可用磁盘空间；某些功能需要额外磁盘空间
  • 某些功能要求使用 iOS 11 或更新版本或 macOS Mojave
  • 某些功能要求使用 Apple ID；须遵守服务条款
  • 某些功能需有兼容的互联网服务供应商；可能需要付费
  • 某些功能需要加入相关计划 (sysin)
  • 某些功能仅适用于部分国家和地区

具体系统版本要求详见下载。

了解 macOS 服务器 5.7.1 中引入的变化

Apple 会继续开发和支持 macOS 服务器，这包括 Open Directory、描述文件管理器和 Xsan 管理。每次用户安装 macOS High Sierra 和更高版本时，系统都会捆绑安装缓存服务器、文件共享服务器和时间机器服务器等最常用的服务器功能，因此更多客户可以免费使用这些重要服务。

从 macOS 服务器 5.7.1 开始，Apple 不再将日历服务器、通讯录服务器、邮件服务器、DNS、DHCP、VPN 服务器和网站等开源服务与 macOS 服务器捆绑在一起。客户可以直接从开源提供商处获取这些服务。这样，当有最安全和最新服务推出时，macOS 服务器的客户可以即刻安装这些服务。

Apple 已发布相关文稿来协助实现这次过渡。

版本记录

更多详见：Mac App Store

5.12 - 2021年12月8日

  • Xsan 管理已从 macOS 服务器中移除。从 macOS Big Sur 开始，macOS 中附带 Xsan 管理的命令行工具。若要进一步了解，请参阅 https://developer.apple.com/support/downloads/Xsan-Management-Guide.pdf。

5.11.1 - 2021年4月28日

  • 改进了“描述文件管理器”性能

  • 新增“未配对的外部启动进入恢复模式”和“使用 Apple Watch 解锁”的访问限制

  • macOS Big Sur 现支持 Xsan 管理。若要进一步了解迁移到新工具，请参阅 https://developer.apple.com/support/downloads/Xsan-Management-Guide.pdf。

5.11 - 2020年12月14日

  • iOS/iPadOS

    • 自动设备注册：在“设置助理”中跳过“已完成恢复”和“已完成更新”面板

    • 安装不可移除的被管理 App

    • Exchange ActiveSync：覆盖之前的密码

    • 网络使用率规则：配置 SIM 规则

    • 通知：允许在锁定屏幕上显示通知预览

    • 访问限制：允许轻 App

    • Wi-Fi：停用关联 MAC 随机化

    • 在设备信息中包括 eSIM 标识符“EID”

  • iPadOS

    • 教育：允许仅配置“共用的 iPad”登录窗口，而非“课堂”

    • 共用的 iPad：管理基于配额的用户

    • 共用的 iPad：停用临时会话

  • macOS

    • 配置“自动前进”

    • 自动设备注册：在“设置助理”中跳过“辅助功能”面板，在设备安全性信息中显示“服务器激活锁忽略码”

    • 安装被管理的 App

    • 帐户配置：被管理本地用户的短名称

    • 关联域：启用直接下载

    • 内容缓存：自动启用网络共享缓存，发送 ContentCachingInformation 命令

    • 访问限制：允许 App 获取文件提供程序信息，推迟 App 更新

  • iOS/iPadOS/macOS

    • VPN：配置新系统扩展和网络配置密钥

    • iOS/Apple tvOS

    • 在被监督的设备上设定时区

5.10 - 2020年4月1日

描述文件管理器

iOS

  • 停用“共用的 iPad”临时用户

macOS

  • 设备注册：支持首位用户登录时启用文件保险箱

  • 查询内容缓存信息

iOS 和 macOS

  • 内容缓存：自动激活互联网连接共享

  • VPN：为“自定 SSL”连接类型配置“提供程序指定要求”

  • VPN：为 Cisco、Juniper、Pulse、F5、SonicWall、Aruba、CheckPoint 和“自定 SSL”连接类型配置网络选项

  • 通过 VPN 发送所有流量

  • 排除本地网络
