Aria Automation 的新功能特性

跨云环境和数据中心实现一致的自动化

VMware Aria Automation 旨在提供自助式云服务、监管式多云自动化以及基于 DevOps 的基础架构管理和安全性 (sysin)，可帮助提高 IT 敏捷性、生产力和效率，从而让您为迎接企业未来做好准备。

  • 缩短销售就绪时间

    采用高级工作流和敏捷模板，减少了手动任务。

  • 易于使用

    通过直观的自助式使用体验 (sysin)，在整个生命周期中快速设置和管理多云环境。

  • 安全性和控制力

    制定跨多云环境保持一致的策略，并通过原生合规性管理、灵活的云保护措施和漏洞修复来增强基础架构。

  • 加快创新速度

    借助基于基础架构即代码和 DevOps 原则构建的面向未来的平台 (sysin)，加快采用新的云原生技术。

VMware Aria Automation 功能特性

  • VMware 的不限云类型的模板

    借助基础架构即代码，使用单个云模板在 VMware Cloud 或任何主要公有云（AWS、Azure、Google Cloud 等）上进行部署。

  • 集中式策略和监管

    通过模板化云和策略定义、云环境可见性和自动化修复功能来管理多个云。

  • 可延展性和自定义设置

    通过 Aria Automation Orchestrator (vRealize Orchestrator)、基于操作的可延展性 (ABX) 以及与常见第三方工具的内置集成，实现完整的可延展性和自定义设置。

  • 配置管理

    通过直观的配置自动化、漏洞修复和合规性实施 (sysin)，实现对虚拟化环境和云环境的初始和后续控制。

  • 自助式多云

    采用统一且一致的 IaaS 使用层、自助目录和幂等 REST API，让用户能够跨云环境请求和置备基础架构资源。

  • 基础架构流水线发布

    访问专门为基础架构应用场景中的 CI/CD 量身定制的、便于用户使用的发布自动化流水线。

应用场景

  • 基础架构的 DevOps
    实现功能强大的基础架构即代码平台 (sysin)，支持基础架构流水线发布和迭代开发。

  • Kubernetes 自动化
    自动管理 Kubernetes 集群和 Namespace，同时支持 vSphere with Tanzu。
    ​ 阅读博客

  • 安全运维
    利用事件驱动型自动化功能，实施全面的闭环 IT 系统合规性实施并修复漏洞 (sysin)。

  • 自助式多云
    跨 VMware Cloud 和公有云提供一致的自助服务使用和基础架构交付服务。

  • 多云环境监管
    使用“一切即代码”方法大规模管理多云环境的成本、性能、安全性、网络连接和配置。

比较各个版本

  • Standard Plus
    基础版本，包含状态管理和后续自动化功能。

    • vRealize Automation SaltStack Config

  • Advanced
    功能齐全的版本，提供集中的策略和控制

    • Standard Plus 版本中的所有功能特性
    • vRealize Orchestrator 和基于操作的可延展性 (ABX)
    • Service Broker 和 Cloud Assembly
    • VMware Cloud Templates (sysin)

  • Enterprise
    提供额外的集成和多云支持

    • Advanced 版中的所有功能特性
    • 原生公有云端点
    • DevOps、Code Stream 和 Kubernetes（包括 Tanzu 集成）
    • 第三方配置工具集成 (sysin)

  • 云环境
    Enterprise 版，以服务形式交付

    • Enterprise 版中的所有功能特性

适用的 VMware 软件下载

建议在以下版本的 VMware 软件中运行（Linux OVF 无需本站定制版可以正常运行，macOS 虚拟化如果不是 Mac 必须使用定制版才能运行，Windows OVF 需要定制版才能启用完整功能）：

  • Server：VMware ESXi 8.0 or with driver & vCenter Server 8.0，ESXi 7.0 or with driver & vCenter Server 7.0
  • macOS：VMware Fusion 13
  • Linux：VMware Workstation 17 for Linux
  • Windows：VMware Workstation 17 for Windows
