简介

一套工具。任何基于 Arm 的项目。

从微型现成微控制器到定制集群处理器的高效嵌入式软件开发

Development Studio 专为 Arm 架构设计，是市场上最全面的嵌入式 C/C++ 专用软件开发解决方案 (sysin)。Development Studio 可加速软件工程，同时帮助您构建强大而高效的产品。

​

Arm Development Studio 包括：

  • Arm 调试器和 Keil µVision 调试器
  • 嵌入式 C/C++ Arm 编译器，包括向后兼容的许可证
  • 流线型性能分析器，用于在 Linux、Android 或裸机上进行系统范围的优化
  • 适用于 MCU 且符合 CMSIS 标准的免版税中间件模块
  • Armv7 和 Armv8 固定虚拟平台用于软件开发的
  • 图形调试器与 OpenGL ES、Vulkan 和 OpenCL 兼容的

版本

  • Bronze Edition 青铜版

    此入门级版本包括适用于所有 Cortex-M 项目的完整工具链，以及适用于选定 Cortex-A 和 Cortex-R 内核的完整调试。为所有受支持的内核提供固定虚拟平台。

    该版本采用 Keil MDK Essential Edition，因此不包含中间件。

  • Silver Edition 银版

    Silver 包含用于在 Cortex-R 和 Cortex-A 系列以及整个 Cortex-M 系列中调试、编译、建模和性能分析的 Arm 内核直至 Armv7 架构的所有基本工具。对于基于 Armv8 Cortex-A 和/或 Cortex-R 架构的系统 (sysin)，此版本还支持调试和性能分析——这对于在这些内核上运行 Linux 的用户来说是理想的选择。

    Silver 还具有用于专用 MCU 开发的 Keil MDK Professional，其中包括完整的中间件。

  • Gold Edition 黄金版

    黄金版非常适合那些为任何 Arm CPU 和架构（包括 Armv8 处理器）上的所有软件开发项目寻求全面工具链的人。调试器、编译器、性能分析工具和虚拟原型支持所有内核。

    此版本还具有用于专用 MCU 开发的 Keil MDK Professional，包括完整的中间件。

  • Platinum Edition 白金版

    白金版是为 Arm 合作伙伴保留的，在最新 IP 可用于设备之前开发。它包括 Gold 的所有功能，但支持 Arm 最新宣布的 IP。

    请 联系 Arm了解更多信息。

系统要求

Hardware requirements

To install and use Arm Development Studio, your workstation must have at least:

  • A dual core x86 2GHz processor (or equivalent).
  • 2GB of RAM.
  • Approximately 3GB of hard disk space.

To improve performance, Arm recommends a minimum of 4GB of RAM when you:

  • Debug large images.
  • Use models with large simulated memory maps.
  • Use Arm Streamline.

Host platform requirements

Arm Development Studio supports the following host platforms:

  • Windows 10
  • Windows 11
  • Red Hat Enterprise Linux 8 Workstation
  • Ubuntu Desktop Edition 20.04 LTS
  • Ubuntu Desktop Edition 22.04 LTS

Note:

Arm Development Studio only supports 64-bit host platforms.
