VMware by Broadcom 解决方案简介

正如 2023 年 12 月 11 日宣布的 那样，VMware 在过去两年的精简和简化其产品组合以及从永久许可到订阅模式的过渡过程中达到了一个新的里程碑。VMware Cloud Foundation 和 VMware vSphere Foundation 现在是我们的两个新产品，仅作为订阅软件提供 (sysin)。我们还提供可选的高级附加组件，可与这些解决方案一起使用。

由于产品组合简化，许多 VMware 软件解决方案将仅作为 VMware Cloud Foundation (VCF) 或 VMware vSphere Foundation (VVF) 的一部分提供。它们不能作为独立的单点解决方案购买。

为了清楚起见，下面概述了我们对产品组合所做的改变：

  • VMware Cloud Foundation 是 VMware 为寻求利用全堆栈基础架构价值的客户推出的全新解决方案，该平台为 vSphere、vSAN 和 NSX 提供完整的 Aria 管理和编排套件以及新服务。
  • VMware vSphere Foundation 是 VMware 针对传统 vSphere 环境中的数据中心优化的解决方案。除了 Aria Operations 和 Aria Operations for Logs 之外，它还包括 Tanzu Kubernetes Grid，作为其标准功能套件的一部分 (sysin)。此外，对基本硬件整合或极少数服务器上的虚拟化等要求较轻的客户仍然可以购买 vSphere Standard 和 vSphere Essentials Plus Kit 的订阅。
  • VMware 附加服务 可用于 VMware vSphere Foundation 和 VMware Cloud Foundation，以进一步增强客户旅程并解决额外的存储、安全、灾难恢复、生成 AI 和其他特定用例。

这种产品组合简化将使客户能够从 VMware 投资中获取更多价值，并使 VMware 能够加速新创新的交付并简化客户的部署和管理。

此外，VMware by Broadcom 还计划提供 “自带许可证” 功能 (sysin)，允许客户从 Broadcom 购买 VMware Cloud Foundation 订阅，并将这些订阅灵活部署到经过 VMware 验证的混合云端点以及他们自己的本地数据中心。

此次过渡的另一个结果是，下表中列出的产品 现已作为独立产品停止供货 (EOA)。我们已注意到，某些产品可能仍可作为 VMware Cloud Foundation 或 VMware vSphere Foundation 产品的一部分供使用。

正如 博客 中详细介绍的，VMware 还宣布了 Aria SaaS 服务的 EoA，但将继续为当前使用 Aria SaaS 服务的客户提供支持，直至其订阅期限结束。订阅期限结束时，客户将需要转换到 VCF 或 VVF。另请注意，Aria SaaS 服务处于维护模式，这意味着不会发布新功能，但在有效订阅期内将提供安全和关键更新。

本公告涵盖所有许可选项，包括永久、支持和订阅 (SnS)、SaaS / 托管和其他订阅类型。它还涵盖了每种产品的每个版本、套件和定价指标。此承保范围的任何例外情况将单独指定。
