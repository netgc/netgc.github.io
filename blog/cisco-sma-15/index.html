跨多个思科集中管理和报告功能 ® 电子邮件安全设备 (ESA) 和思科网络安全设备 (WSA) 以及思科内容安全管理设备 (SMA)。Cisco SMA 与 Cisco ESA 和 WSA 的集成简化了电子邮件和 Web 安全的规划和管理，改进了合规性监控，使可接受使用策略的一致执行成为可能，并增强了威胁防护。

组织通常必须在人员和预算有限的情况下协调地理分散的团队中的多个设备的管理和管理。Cisco SMA 基于针对报告和跟踪而优化的强大平台而构建，可提供高性能和可扩展性以及行业领先的保护和控制，从而实现长期投资价值。

特点和优点

Cisco SMA 的特性和优点将在以下各节中讨论，并在表 1 中进行更详细的描述。

简化管理和规划

使用 Cisco SMA 易于使用的直观界面，网络管理员可以从单个控制台向多个 Cisco ESA 和 WSA 发布策略设置和配置更改。或者，组织可以将特定设备专用于单个应用程序以进行大批量部署。

此外，当安全设备超出其推荐容量时，网络管理员会收到通知。Cisco SMA 报告每秒的事务数以及系统的延迟、响应时间和代理缓冲存储器。此信息使网络管理员能够确定何时需要重新配置系统或安装其他设备。

改进合规监控和执行

集中报告和跟踪有助于确定哪些用户违反了可接受的使用政策，识别任何部门或网站的政策违规行为，并监控 Facebook 和 YouTube 等 Web 2.0 应用程序的使用情况以及对特定类别 URL 的访问，例如 “赌博” 或“体育”。

通过集中管理多个设备，管理员可以在整个组织内实施一致的可接受的使用策略。

增强的威胁防护

Cisco SMA 提供组织安全运营的全面视图，提供更好的威胁情报、防御和补救。重要功能包括集中管理电子邮件垃圾邮件隔离、跨多个 Web 安全网关的全面威胁监控、Web 信誉评分和僵尸网络检测。Cisco SMA 的报告功能还可用于帮助管理员识别和解决涉及数据丢失防护 (DLP) 的活动。

高性能和可扩展性

Cisco SMA 有两个针对报告和跟踪而优化的专有数据库，而不是单个通用数据库。对每个查询应用适当的计算，以快速生成实时报告。

基于高性能 Cisco AsyncOS 构建 ® Cisco SMA 操作系统提供业界领先的可扩展性，以满足小型、中型和大型企业以及服务提供商的需求。

使用思科内容安全管理虚拟设备进行灵活部署

思科内容安全管理虚拟设备 (SMAV) 显着降低了管理电子邮件和 Web 安全的成本，尤其是在高度分布式网络中。您的网络管理员可以使用您现有的网络基础设施在需要的地方和时间创建实例。Cisco SMAV 是 Cisco SMA 的软件版本，在 VMware ESXi 虚拟机管理程序和思科统一计算系统之上运行 ™ （思科 UCS ® ）服务器。购买任何思科电子邮件或网络安全软件包的 SMA 软件许可证，您将获得无限数量的思科 SMAV 实例。

借助 Cisco SMAV，您可以通过简化的容量规划即时响应日益增长的流量增长。您不需要购买和运输设备，因此您可以支持新的业务机会，而无需增加数据中心的复杂性或雇用额外的员工。

为什么选择思科？

安全对于您的网络来说比以往任何时候都更加重要。由于威胁和风险持续存在，加上对保密性和控制的担忧，安全对于提供业务连续性、保护有价值的信息和维护品牌声誉是必要的。嵌入网络结构中的思科安全解决方案使您能够以高度安全的方式连接到正确的信息，而不会中断您的业务。没有任何组织像思科那样了解网络安全。我们的市场领导地位、行业领先的威胁防护和预防、创新产品和长寿命使我们成为满足您安全需求的合适供应商。
