1. OVF 版本

2. 系统账号

开启 root 账号，密码通过部署向导设定

3. 分区

4. 系统配置

  1. 格式化网卡命名 eth0，IP 默认手动配置（通过部署向导配置）

  2. 使用阿里云镜像源

  3. 安装常用软件
    sudo apt install zip unzip
    sudo apt install lrzsz
    sudo apt install htop #20.04已默认安装
    sudo apt install lnav
    sudo apt install fd-find #fd命令，20.04已经收录
    sudo apt install ripgrep #rg命令，20.04已经收录
    sudo apt install tree
    sudo apt install build-essential #Following command will install essential commands like gcc, make etc.
    sudo apt install net-tools #ifconfig、netstat、route等命令集

  4. NTP 使用 chrony，使用默认配置

  5. 安装 SNMP，默认配置

  6. 20.04 自带 vm-tools 版本为 11.0.5，与 ESXi 7.0.0 匹配

其他版本配置同，增加/目录容量

下一个版本什么时候更新

预计 Ubuntu 20.04.5 版本发布时，将同步更新。

适用的 VMware 软件下载

建议在以下版本的 VMware 软件中运行：

  • Server：VMware ESXi 7.0 U1 定制版镜像

  • macOS：VMware Fusion 12

  • Linux：VMware Workstation 16 for Linux

  • Windows：VMware Workstation 16 for Windows
