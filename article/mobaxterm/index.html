欢迎使用 MobaXterm, Windows 系统中的 Xserver 和 SSH 客户端

MobaXterm：X server 和 SSH 客户端

MobaXterm 是您 “远程计算的旗舰工具箱” 。在单个 Windows 应用程序中，它提供了许多功能，这些功能是为程序员，网站管理员，IT 管理员以及几乎所有需要以更简单的方式处理其远程作业的用户量身定制的。

MobaXterm 提供了所有重要的远程网络工具（SSH，X11，RDP，VNC，FTP，MOSH 等）和 Windows 桌面上的 Unix 命令 （bash，ls，cat，sed，grep，awk，rsync，…），它们整合在一个开箱即用的单个可便携 exe 文件中。

为您的远程任务使用 “多合一网络应用程序” 有很多优点，例如当您使用 SSH 连接到远程服务器时，将自动弹出图形化的 SFTP 浏览器以直接编辑您的远程文件。您的远程应用程序也将使用内置的 X 服务器在 Windows 桌面上无缝显示。 查看演示

您可以免费下载和使用 MobaXterm Home Edition。如果要在公司内部使用它，则应考虑订阅 MobaXterm 专业版：这将使您可以使用更多功能和“定制程序” 软件。 功能比较

在开发 MobaXterm 时，我们专注于一个简单的目标：提出一个“直观的用户界面”，以便您通过不同的网络或系统“有效地访问远程服务器”。 屏幕截图

功能介绍

MobaXterm 选项卡式终端

在此控制台窗口中，将允许您运行所需的 Unix 命令：ls，cd，grep，awk，tail，cut，sed，wget，rsync，telnet，ssh，rlogin，rsh… 所有必不可少的 Unix 工具存在。如果您需要基本 MobaXterm 文件中没有的程序，则可以下载免费的插件。

会话管理

MobaXterm 允许您启动远程会话。您可以选择创建 SSH，Telnet，Rlogin，RDP，VNC，XDMCP，FTP，SFTP 或串行会话。您启动的每个会话都会自动保存并显示在左侧边栏中。

图形化 SFTP 浏览器

当您使用 SSH 登录到远程服务器时，图形化的 SFTP 浏览器会在左侧栏中弹出。它允许您使用安全的 SFTP 连接直接从远程服务器通过拖拽上传或者下载文件。

X11 服务器

当您运行 SSH，TELNET 或 RLOGIN / RSH 会话时，您将能够直接在本地 Windows PC 上显示远程应用程序。在 SSH 会话中，无需设置 “DISPLAY” 变量，因为 MobaXterm 使用 X11 转发来简化和保护您的工作。

增强的 X 扩展

基于 X.org 的内置 X 服务器提供了最新 X 服务器实现中可用的最新功能：包括诸如 OpenGL，Composite 或 Randr 的扩展。还支持 Xdmcp 协议。

多次执行

编写一次，随处执行：使用此工具，您可以同时在许多不同的服务器上执行相同的命令。

嵌入式服务器

MobaXterm 允许您启动网络守护程序以进行远程访问。不需要额外的工具或运行时即可使用这些守护程序。

内置工具

MobaXterm 为系统管理员，开发人员，网站管理员和需要有效使用计算机的所有用户提供了一些有用的工具。

远程 Unix 桌面（XDMCP）

借助 MobaXterm，您可以在远程 Solaris 桌面上工作，就像使用 XDMCP 协议在 Windows 计算机上本地工作一样。

远程 Windows 桌面（RDP）

使用 RDP 协议控制您的远程 Windows 计算机 / 服务器。会话管理器中提供了许多 RDP 配置设置。

SSH 网关

在 ssh，telnet，RDP，VNC 会话中，您可以选择一个 “SSH 网关”（又称 “跳转主机”），以告诉 MobaXterm 先连接到 SSH 服务器，然后再连接到要访问的终端服务器。这使您可以访问防火墙后面的某些服务器并保护连接。

SSH 隧道（端口转发）

图形 SSH 隧道管理器允许您使用直观的图形工具创建 SSH 隧道。

MobApt 软件包管理器

MobaXterm 软件包管理器（MobApt / apt-get）允许您直接在 MobaXterm 终端中下载并使用更多的 Unix 工具。

文本编辑器

使用 SSH 连接到远程服务器时，只需双击嵌入式文件编辑器 “MobaTextEditor”，就可以直接编辑远程文件！

支持宏

您可以在 MobaXterm 终端中记录宏：您在终端中键入的所有内容都会被记录下来，以便以后在其他服务器上重放。

密码管理

MobaXterm 能够保存您的会话密码，并通过使用 “主密码” 来保护它们的安全。

终端中的语法高亮显示

即使您连接到不支持颜色的远程服务器，MobaXterm 嵌入式终端也可以高亮显示语法或为关键字使用颜色。这在分析日志文件，编辑代码或尝试检测重要的关键字时特别有用。您甚至可以轻松地构建自己的语法定义。

专业定制

MobaXterm 专业版使您可以访问 “定制程序” 软件。该程序允许您使用自己的徽标和默认设置生成 MobaXterm 的自定义副本。
