VMware vRealize Log Insight 8.2

概述

vRealize Log lnsight 提供了高度可扩展的异构日志管理功能，它具有多个可在其中执行操作的直观仪表盘、完善的分析功能和范围更广的第三方延展性。它还能够跨物理、虚拟和云计算环境提供深入的运维洞察信息并加快故障排除速度。

VMware vRealize Log Insight 提供集中式日志管理、深度运维可见性和智能分析，从基础架构到跨本地部署和私有云环境的应用，都能提供更好的故障排除和更高的安全性。

vRealize Log Insight 如何提供更好的日志管理？

  • 更好地进行快速故障排除和根本原因分析

    跨物理、虚拟和云计算环境提供创新的索引编制和基于机器学习的智能分组功能，以实现高性能搜索，从而加快故障排除速度。

  • 大规模监控和管理计算机数据

    分析大量日志数据，提供接近实时的监控、搜索和日志分析功能，并配备一个可查看存储的查询、报告和警报的仪表盘。跨整个 IT 环境快速关联事件。

  • 从非结构化数据创建结构

    收集和自动识别机器生成的所有类型的日志数据结构（如应用日志、网络跟踪、配置文件、消息、性能数据、系统状态转储等），从而为执行分析构建高性能索引。

  • 降低运维开销

    提供包括无限数据量的可预测定价模型，不要求基于峰值用量和最坏情形购买许可证。

功能特性

  • 通用日志收集和分析

    利用 vRealize Log Insight 收集和分析机器生成的所有类型的日志数据。管理员可将其应用于环境中的所有内容（操作系统（包括 Linux 和 Windows）、应用、存储、防火墙、网络设备等），从而通过日志分析获得整个企业范围的可见性。

  • 企业级可扩展性

    高度可扩展，并且专为处理机器生成的所有类型的数据而设计。最近的内部测试发现，Log Insight 在针对 10 亿条日志消息的查询测试中比其他业内领先的解决方案快 3 倍。每个节点接收的数据量翻番，并且每个节点每秒最多可支持 15,000 个事件。

  • 直观的图形用户界面、轻松部署

    借助基于 GUI 的直观界面，用户可以轻松运行简单的交互式搜索以及深入的分析查询，快速获得洞察信息，从而能够即时提供价值并提高 IT 效率。vRealize Log Insight 自动为您的数据选择最佳显示方式，从而节省您的宝贵时间。

  • 内置的 VMware SDDC 知识库

    由 VMware 专家开发的 vRealize Log Insight 附带内置的知识库和对 VMware SDDC 技术的支持。但是，您可以分析虚拟基础架构之外的日志，并使用中央日志管理解决方案来分析您的整个 IT 环境中的数据。

  • 与 vRealize Operations 集成

    与 vRealize Operations 平台的集成可将运维可见性和主动管理功能延展到基础架构和应用。这种集成还可将非结构化数据（例如日志文件）与结构化数据（例如衡量指标和关键绩效指标）融合在一起，帮助您最大限度提高投资回报。

新增功能

以下是 vRealize Log Insight 8.2 的一些主要功能亮点，可帮助您比以往更快、更准确、更有效地利用日志数据：

  • 分区存储使用情况和最早的日志载入时间：对于每个数据分区，现在会显示有关已用存储和首次日志载入时间的信息，以及筛选条件、保留期限、是否已启用分区等其他详细信息。有关已存储数据使用期限的信息有助于满足在定义的时间段内（通过保留期限进行配置）保留日志事件这一合规性要求。

  • 纵向扩展：与“大型”大小相比，引入的“超大型”这一预设虚拟机大小具有双倍的计算资源，从而有助于更有效地扩展和增强大型环境的性能。有关详细信息，请参见 https://kb.vmware.com/s/article/80928。

  • 通过 SSL 进行 syslog 事件转发：使用 syslog 进行事件转发的功能得到了增强，现在可以选择通过 SSL 保护连接。

  • 无限制的日志导出管理：能够在导出完成时发送电子邮件通知，并跟踪导出任务的进度。如果任务已排入队列，您可以查看该任务在队列中的位置。

  • IPv6 零压缩支持。

  • 强化本地帐户安全性：

    为本地帐户用户新增了可选的密码策略限制，具体包括以下附加规则：

    • 密码必须包含至少 15 个字符。
    • 用户每 24 小时只能更改一次密码。
    • 用户更改密码时，无法使用最近五次用过的密码。
    • 用户更改密码时，新密码必须至少有八个字符与旧密码不同。
    • 如果在 35 天内未登录或在 60 天内未更改密码，则将锁定用户帐户。

  • 对基于已提取字段的查询进行增量性能改进。

  • 内容包搜索：无需离开产品 UI 即可在内容包商城中搜索内容包。

  • 内容包更新：

    • VMware NSX Advanced Load Balancer 1.0（新）
    • VMware Cloud Director Availability 1.0（新）
    • VMware Horizon 4.0
      • VMware Horizon 7.x 和 BLAST 桌面协议支持
      • 新的“常规 - 性能”和“常规 - 可用性”仪表板
    • NSX-T 3.9（支持 NSX-T 3.0 和 3.0.1）
    • vRealize Suite Lifecycle Manager 8.0.1 及更高版本（支持 vRealize Suite Lifecycle Manager 8.1）
    • vRealize Operations Manager 4.1（支持 vRealize Operations Manager 8.2）
    • vSAN 2.2
      • “概览”仪表板和“存储策略事件”仪表板
      • 支持 vSAN 7.0
