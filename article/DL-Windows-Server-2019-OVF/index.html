请注意：

1.1 此处密码必须符合复杂性要求（系统默认值）。

密码必须符合下列最低要求:

  • 不能包含用户的帐户名，不能包含用户姓名中超过两个连续字符的部分
  • 至少有六个字符长
  • 包含以下四类字符中的三类字符:
    • 英文大写字母(A 到 Z)
    • 英文小写字母(a 到 z)
    • 10 个基本数字(0 到 9)
    • 非字母字符(例如 !、$、#、%)
  • 在更改或创建密码时执行复杂性要求

1.2 输入任意字符将取消 sysadmin 账号

2.2 子网掩码前缀使用 CIDR 表示法，例如掩码 255.255.255.0 填写 24

OVF 特性

1. OVF 版本

  1. VM 版本 11，即兼容 ESXi 6.0 / Fusion 7.x / Workstation 11.x / Player 7.x 及以上版本
  2. VM-Tools 版本：11.2.5 (2021-01-12)
  3. 启用 CPU、内存热添加

2. 系统配置

  1. 安装时选择时间和货币格式：Chinese (Simplified, China)，键盘和输入方法：简体中文–美式键盘（US）
  2. 系统版本：Windows Server 2019 Datacenter, with DesktopExperience
  3. 分区：System Reserve（549M），C 盘 40G 整数分区
  4. 启用远程桌面
  5. 启用 ICMPv4 防火墙规则
  6. 新建了一个名为 sysadmin 的管理员账号（特殊用途），部署时候可以禁用该项
  7. 添加了 “Command Prompt” 和 “PowerShell” 右键快捷命令菜单

3. 简体中文界面支持

本次英文版添加了简体中文界面语言支持，切换方法如下：

Settings > Time & Language > Language, Windows Display Languages 下拉勾选 ”Chinese (Simplified, China)”

注销登录或者重启生效

4. 系统激活

SLIC 2.5 OEM 激活，使用 Dell 证书。使用定制版的 VMware 软件 自动激活。

5. 系统组件

  1. 添加组件：Telnet Client、SNMP-Service、Windows Backup
  2. 添加组件：.NETFX 3.5 (启用此组件需要系统 iso 镜像，这里预先配置)

6. 远程桌面性能优化

  1. 禁用 Logon Backgroud
    echo 个性化设置：禁用登录背景 Logon Image
    reg add “HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Windows\System” /v DisableLogonBackgroundImage /t REG_DWORD /d 1 /f
  2. 更换 Lock Screen 为默认蓝色
    echo 个性化设置：禁用设置中 Lock Screen 功能
    reg add “HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Personalization” /v NoLockScreen /t REG_DWORD /d 1 /f
  3. 替换 Desktop Backgroud 为灰色

7. 应用软件

  1. AutoRuns 13.90、7-Zip 19.00、Unlocker 1.9.2、Attribute Changer 10.00a（全部为免费软件）
  2. 预装了 Microsoft Edge 88.0.705.50

8. 个性化配置

  1. 用户配置

  2. 几个关于隐私和使用记录的配置，通过 HKEY_LOCAL_MACHINE 进行配置

9. 备注

关于 OVF 版本的兼容性：

  • 支持的部署方式

1、通过 ESXi 6.5 及以上版本的 Host Client 进行部署。
2、通过 vSphere Web Cleint（Flash）或者 vSphere Client（H5）6.5 及以上版本进行部署。该种方式可以运行在 ESXi 6.0 的主机上。

  • 不支持的部署方式

1、不支持通过 VMware vSphere Client 6.0 桌面 C# 客户端进行部署，该客户端不支持 SHA256。
2、不支持通过 ESXi 6.0 的 Host Client 进行部署，该版本的 Web 客户端不支持自定义 OVF 配置文件，也存在严重的浏览器兼容性问题。
